# Gama
## ServicoAlunos

** Baixe para sua maquina, sera necessario ter o docker instalado **

### Inicie o banco de dados:
acesse a pasta do projeto e digite:
docker compose up -d

### Compile o projeto 
acesse a pasta servicoAlunos atraves de um terminal e digite:
dotnet build 
dotnet run

### Acesse o Swagger
abra um browser e digite:
http://localhost:5055/swagger/index.html
